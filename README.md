## Usage

Add following link tag to index.html for using the components.

```html
<link rel="stylesheet" href="https://min-ui.netlify.app/src/css/minui.css" />
```

### Running in local

Clone the project

```shell
git clone https://github.com/Pratham82/min-ui
```

Navigate to the project

```
cd min-ui
```
